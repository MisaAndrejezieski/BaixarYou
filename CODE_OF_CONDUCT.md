# Código de Conduta do Contribuidor

## Nosso Compromisso

Como participantes deste projeto, nos comprometemos a tornar a participação uma experiência livre de assédio para todos.

## Nossos Padrões

Comportamentos que contribuem para um ambiente positivo:

- Usar linguagem acolhedora e inclusiva
- Respeitar pontos de vista diferentes
- Aceitar críticas construtivas

Comportamentos inaceitáveis:

- Uso de linguagem sexualizada
- Ataques pessoais ou insultos
- Assédio público ou privado

## Aplicação

Casos de comportamento abusivo podem ser reportados ao mantenedor.
